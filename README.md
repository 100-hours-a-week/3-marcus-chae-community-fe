# Community Frontend
Vanilla JavaScript 기반 커뮤니티 웹 애플리케이션

## 개발 기간

2025.9.15 ~ 2025.12.07

## 기능

- 회원가입/로그인 (JWT 토큰 관리)
- 게시글 작성/수정/삭제/조회
- 댓글 작성/삭제
- 게시글 목록 조회 (무한 스크롤)
- 조회수 및 댓글 수 표시
- 사용자 프로필 관리

## 기술 스택

- Vanilla JavaScript (ES6+)
- HTML5, CSS3
- Express.js (개발 서버)
- HTTP Proxy Middleware
- ESLint, Prettier

## 프로젝트 구조

```
community-fe/
├── public/
│   ├── css/          # 스타일시트
│   ├── js/           # JavaScript 모듈
│   └── pages/        # HTML 페이지
├── server.js         # Express 개발 서버
└── package.json      # 프로젝트 설정
```
